# Organizador-de-Arquivos


Script em Python que organiza arquivos em uma pasta em categorias como:
- Fotos
- Vídeos
- Documentos
- Banco de Dados
- Programação Web
- Programação Script e Automação

## Como usar:
1. Defina o caminho da pasta:
- camin = "C:/Users/Seu_Usuario/test"

2. Execute o script:
- python3 Organizador.py

## Tecnologia Usadas:
- Python
- Os
- Shutil
